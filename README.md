# 🌊 深海都市诗性故事生成器

**Deep Sea City Poetic Story Generator**  
一个为 WorkBuddy 设计的 AI Skill，能将日常词语转化为深海都市的神话与童话。

---

## 🐋 这是什么？

这不是普通的新闻改写器。  
它是一只游弋在深海第三层窗外的鲸鱼——  
专门收集人类丢下的词语碎片，  
用深海盐分和月光石粉慢慢包裹，  
最后吐出一个完整的、带着磷光的故事。

你的每个日常词语，  
都可能被它改写成：

- 📜 **创世神话**（关于面包如何成为深海货币的起源传说）
- 🏰 **超现实童话**（珊瑚宫殿里会流泪的盐晶精灵）
- 🐟 **鱼怪寓言**（正经派鱼怪公司的忧郁职员日常）
- 🌌 **星际蓝调**（可可暗流与反重力糖霜的相遇故事）

## ✨ 特色功能

### 1. **两种创作模式**
- **词语模式**：将单个词语（如“面包”、“冰红茶”）转化为 500 字左右的深海神话
- **冒险模式**：将短语或场景转化为 800 字左右的超现实故事，融入完整情节与人设

### 2. **优先示例风格**
本 Skill 优先采用 **用户创作的《可可暗流与面包的星图跃迁》** 作为核心示例，  
展示“从创世神角度思考、物件精灵化、意象奇特、超现实风格”的创作原则。

### 3. **诗意世界观**
深海都市不是冰冷的水下城市，而是一个 **有呼吸、有意识、有记忆** 的活体。  
这里的每个物件都可能突然意识到“我是谁”，  
然后开始用盐粒写诗，或用锈钉讲故事。

## 🏗️ 世界观设定（浓缩版）

### 创世神
- 本体是“裂隙”本身，没有固定形态
- 通过呼吸创造世界：第一口凝成岩石，第二口化作会思考的盐粒，第三口让水意识到“我是水”

## 🎨 设计理念

### 核心创作原则
1. **从创世神角度思考**：每个物件都可能突然觉醒，意识到自己的存在
2. **物件精灵化**：巧克力面包会写诗，红茶会打嗝，盐粒会叹息
3. **意象奇特**：酸味可能是“时间不肯愈合的伤口”，甜味可能是“在账户里跳舞的积分”
4. **超现实风格**：逻辑可以打嗝，语法可以发酵，情绪可以用海盐称重

### 叙事节奏
- 🎵 **浪的节奏**：长短句交错，段落间有足够的呼吸感
- 🎭 **意象流转**：不说逻辑，只说意象，让意象自己讲故事
- 💎 **署名传统**：每篇故事结尾都有“——海天一色”的签名
